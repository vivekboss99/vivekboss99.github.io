07Sign Language Recognition

Artificial Intelligence Project

May, 2020

Completed

Project Summary

  • Multi-Class Classification Project.
  • Used Tensorflow 2.0 along with Python 3.
  • Used MNIST dataset for Sign Language Training and Testing datasets.
  • Used Image Augumentations for optimizing the training process and increasing the efficiency,used two convolutional neural network layers and two dense layers for training the image data and ReLu(Rectified Linera Unit) as the activation function.
  • Used sparse categorical crossentropy as loss function and rmsprop(Root Mean Square Propagation) as the optimizer.
